typedef void(^ TVersityAFCommandBlock)(BOOL isError, NSError *error)

Defines the block to be called with the result of an an asynchronous player command.

Parameters
isErrorYES if an error occured. NO if the command was invoked successfully.
errorIf an error occured, error will contain an error object with error information.
The specific error codes this parameter can return are available in the documentation of TVersityAFPlayer methods.
