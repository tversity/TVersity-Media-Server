Detailed Description

Classes representing iOS Device Assets.

Media assets on an iOS device are represented by different object types from different frameworks. For example the Assets Library framework provides access to photos with an ALAsset object. The AV Foundation framework provides access to videos with an AVAsset object, and the Media Player framework provides access to music with an MPMediaItem object.

The classes in this module provide a uniform intefrace for accessing assets from all of these frameworks and using them with the AirFlinger SDK.
