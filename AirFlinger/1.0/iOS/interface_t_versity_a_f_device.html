Detailed Description

An abstract class that provides common information about a device. You can get the current devices by calling getCurrentDevices: (TVersityAFAirFlinger)

Usually this class should not be used directly and instead objects of this type should be cast into one of the decendants classes.

Property Documentation

- (TVersityAFDeviceStatus) currentStatus
readatomicassign

Current device status.

Notice this value does not update dynamically. It represents the status of the device at the moment this instance of the object was created.

To get real time updates on a device status, register to the TVersityAFEventPlayerStatusChange or TVersityAFEventServerStatusChange events.
