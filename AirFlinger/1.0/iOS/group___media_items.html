Detailed Description

Classes representing media objects on a local iOS Device or on a remote media server.

The classes in this module represnt media objects on a local iOS device or on a remote media server and provide a uniform interface for accesing them. Media objects are media items (e.g an image or an audio file) or media containers (e.g. a container object on a remote server).
