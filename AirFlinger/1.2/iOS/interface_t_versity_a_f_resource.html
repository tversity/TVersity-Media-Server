Detailed Description

Represents a single resource of a media object. Ususally, a resource is a file (local or remote) that can be played or displayed in a DMR (Digital Media Renderer). A resource may represent however other kinds of content.

Usually you do not need to create such objects yourself. You get a resource from a media object TVersityAFMediaObject::resources property
